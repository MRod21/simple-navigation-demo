h1. Simple Navigation Sample Application for Sinatra

# Point your console to this directory
# Use @bundle install@ to setup the required gems
# Open the file sample_app.rb and either enable classic style app or modular style app.
# Enter @shotgun sample_app.rb@ (class style app) or @shotgun config.ru@ (modular style app) to start the server
# Goto @http://localhost:9393@ to try out the demo-application
